# BJJ Notizbuch – Sprachnachrichten Web App 

Dieses Projekt ist eine einfache, browserbasierte Web-App, mit der ich nach dem Brazilian-Jiu-Jitsu-Training Techniken als **Sprachnotizen** festhalten kann.  
Die Anwendung funktioniert **offline**, speichert Aufnahmen **lokal im Browser** und benötigt kein Backend.

## Motivation
Nach dem Training möchte ich Techniken schnell und unkompliziert dokumentieren, ohne alles aufschreiben zu müssen.  
Diese Web-App dient als persönliches Notizbuch und als Praxisprojekt zur Anwendung von Web-Technologien.

## Funktionen
-  Aufnahme von Sprachnachrichten über das Mikrofon
-  Zuordnung zu einzelnen Techniken / Notizen
-  Lokale Speicherung im Browser (offline nutzbar)
-  Funktioniert auf Desktop und mobilen Geräten
-  Veröffentlichung über GitHub Pages (HTTPS)

## Offline-Nutzung
Die Anwendung kann nach dem ersten Laden auch **ohne Internetverbindung** verwendet werden.  
Alle Aufnahmen werden lokal auf dem jeweiligen Gerät gespeichert.

## Nutzung
1. Webseite im Browser öffnen
2. Mikrofonzugriff erlauben
3. Sprachnachricht aufnehmen
4. Technik benennen und speichern
5. Notizen später erneut anhören

## Hinweis
Die gespeicherten Audiodateien liegen **nur lokal auf dem jeweiligen Gerät** vor.  
Ein automatischer Upload auf einen Server findet nicht statt.
